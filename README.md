# Chalk

A clean, minimal directory of 500+ free university CS courses from MIT, Stanford, Berkeley, CMU and more.

Based on [Developer-Y/cs-video-courses](https://github.com/Developer-Y/cs-video-courses) — a community-maintained list of Computer Science courses with video lectures.

---

## Overview

Chalk is a single-page front-end for browsing free CS university lectures. No signup, no paywall, no backend. Pick a category and start learning.

Courses span 20+ categories including Machine Learning, Algorithms, Operating Systems, Distributed Systems, NLP, Computer Vision, and more — sourced from openly available lecture series on YouTube and university course websites.

---

## Features

- 500+ courses from top universities and institutions
- Filter by category or search by course title, instructor, or institution
- Direct links to YouTube playlists and external course pages
- Fully client-side — no server, no database, no dependencies
- Responsive design

---

## Tech Stack

Vanilla HTML, CSS, and JavaScript. Single file. No frameworks, no build step, no package manager.

## Usage

Clone the repo and open `index.html` in a browser. No install required.

```bash
git clone https://github.com/yourusername/chalk.git
cd chalk
open index.html
```

Or deploy directly to any static host — GitHub Pages, Vercel, Netlify, Cloudflare Pages.
